# 나디(Naddy) 손익계산 규칙집 v1.0

> **작성일**: 2026-04-14
> **목적**: 디자인+마케팅 에이전시 "나디"의 수익/비용 구조를 명확히 하고, 프로젝트 단위·월 단위로 손익을 정확히 판단하기 위한 내부 규칙 문서.
> **사용법**: 견적 낼 때, 프로젝트 끝날 때, 월말 결산할 때 이 문서를 기준으로 판단한다. 직원 채용·연봉 변동·비용 변화 시 즉시 갱신한다.

---

## 0. 왜 이 문서가 필요한가

지금까지 나디는 **견적 → 수주 → 작업 → 입금** 의 사이클을 반복해왔지만, 단 한 번도 **"이 프로젝트에서 얼마가 남았는가"** 를 정확히 계산한 적이 없다.

그래서 다음과 같은 문제가 발생한다:
- 고객이 "단가가 비싸다"고 할 때 근거로 반박하지 못함
- 어떤 프로젝트가 돈이 됐고 어떤 게 손해였는지 모름
- 직원을 추가하면 매출이 얼마나 더 필요한지 모름
- 연말에 통장 잔고는 있는데 그게 이익인지 미지급금인지 모름

이 문서는 이 혼란을 끝내기 위한 **회사의 회계적 뼈대**다.

## 1. 회사 기본 전제

### 1.1 업종
- 디자인 에이전시 (+ 마케팅 서비스 병행)
- 주 수익 모델: **프로젝트 단위 도급**
- 향후: 월 정기(리테이너) 계약 도입 가능성 있음

### 1.2 인원 구성

| 시점 | 구성 | 비고 |
|---|---|---|
| **현재 (2026.04)** | 대표 1 + 디자이너1 (월 230만) | 2인 체제 |
| **2026.06 예정** | 대표 1 + 디자이너1 (월 230만) + 디자이너2 (월 250만) | 3인 체제로 확장 |

### 1.3 원가 구조 특성
- 에이전시는 **사람의 시간**이 원가의 대부분
- 자재비·재고 없음
- 고정비(간접비) 비중이 상대적으로 큼
- → **"누가 몇 시간/몇 일 투입했는가"가 손익의 거의 전부**

## 2. 📌 규칙 #1 — 1인 일당 원가

### 2.1 공식

```
1인 일당 원가 = (월 세전급여 × 1.183) ÷ 월 근무일 21 ÷ 가동률 0.65
```

- **× 1.183**: 4대보험 회사부담(~10%) + 퇴직금 충당(~8.3%) 포함
- **÷ 21**: 월 평균 실근무일
- **÷ 0.65**: 가동률 — 직원이 하루 8시간 중 실제 프로젝트 작업에 쓰는 비율은 65% (나머지 35%는 회의·교육·내부업무·휴식)

### 2.2 현재 확정된 일당 원가

| 인원 | 월 세전 | × 1.183 = 회사부담 총액 | ÷ 21 ÷ 0.65 = **일당 원가** |
|---|---|---|---|
| 대표 | 3,500,000 | 4,140,500 | **약 303,300원/일** |
| 디자이너1 | 2,300,000 | 2,720,900 | **약 199,300원/일** |
| 디자이너2 (26.06~) | 2,500,000 | 2,957,500 | **약 216,700원/일** |

### 2.3 운영 규칙

1. **모든 인원은 매일 타임시트 작성** — "오늘 어느 프로젝트에 몇 시간"
2. 대표의 영업·관리·교육 시간은 프로젝트에 잡지 않음 (회사 운영비로 흡수)
3. 신입 채용·연봉 변경 시 위 표 즉시 갱신
4. **가동률 65%는 잠정값** — 타임시트 3개월 쌓이면 실측치로 교체
5. 대표 본인도 예외 없이 타임시트 작성 (방법 A)

### 2.4 "원가 일당"의 의미

**원가 일당은 '팔 가격'이 아니라 '회사 금고에서 하루에 빠져나가는 돈'이다.**
- 직원에게 주는 월급이 아니다 (월급보다 높다 — 4대보험/퇴직금 포함)
- 고객에게 청구하는 가격도 아니다 (청구는 2배)
- 이 숫자는 **회사 내부 판단용**이지 외부 공개용이 아니다

## 4. 📌 규칙 #3 — 월 손익분기점 (BEP)

### 4.1 공식

```
월 손익분기 매출 = Σ(월 인건비 총원가) + 월 간접비
```

### 4.2 현재 BEP

| 시점 | 인건비 총원가 | 간접비 | **월 BEP** | 연 환산 |
|---|---|---|---|---|
| **현재 (대표+디자이너1)** | 414 + 272 = 686만 | 230만 | **약 917만원/월** | **약 1.1억/년** |
| **2026.06~ (대표+디자이너 2명)** | 414 + 272 + 296 = 982만 | 230만 | **약 1,212만원/월** | **약 1.45억/년** |

### 4.3 해석

- 이 매출 **미만** = 🔴 적자
- 이 매출 **같음** = 🟡 본전 (이익 0원)
- 이 매출 **초과** = 🟢 이익
- **목표 이익 20% 원할 경우**: BEP × 1.20 = 현재 기준 월 1,100만 / 26.06~ 월 1,454만

### 4.4 현실 인식

> 연 매출 **1.5억을 찍어도 이익은 수백만원 수준**이다.
> "매출"과 "이익"은 완전히 다르다. 감각이 아닌 **숫자**로 관리해야 한다.

## 5. 📌 규칙 #4 — 프로젝트 견적 공식

### 5.1 청구 일당 (Billing Rate)

```
청구 일당 = 원가 일당 × 멀티플라이어 2.0
```

| 인원 | 원가 일당 | **청구 일당 (기본 2.0배)** |
|---|---|---|
| 대표 | 303,300 | **600,000원/일** |
| 디자이너1 | 199,300 | **400,000원/일** |
| 디자이너2 | 216,700 | **430,000원/일** |

### 5.2 왜 2.0배인가

원가 일당 × 2.0 = 대략 다음의 합계:
- 1.0 : 본인 인건비 회수
- +0.3 : 간접비 회수 (임대료·SW·세무·장비 감가 등)
- +0.3 : 가동률 오차 + 무료수정 + 영업손실 완충
- +0.4 : 회사 운영이익 (재투자·위기대비·다음해 기반)

**국내 디자인 에이전시 MM 단가(800만~1,500만)와 일치**하는 건강한 범위의 **하단**에 해당.

### 5.3 멀티플라이어 조정 기준

| 상황 | 멀티플라이어 | 근거 |
|---|---|---|
| **표준 프로젝트** | 2.0배 | 기본 |
| **장기·대형 고객 (3개월+)** | 1.8배 | 안정적 매출, 가동률 보장 |
| **공공기관·대기업** | 2.0~2.2배 | 지급 안정성 높음 |
| **단기·급한 프로젝트 (1~2주)** | 2.3배 | 가동률 리스크, 일정 희생 |
| **새 고객·신뢰 불명** | 2.2배 | 대금 리스크 프리미엄 |
| **초단기·당일 작업** | 2.5배+ | 다른 일정 밀어내야 함 |
| **1.5배 미만** | ❌ **금지** | 간접비도 회수 못함 |

### 5.4 견적 공식

```
견적가 = Σ(투입인원 × 투입일수 × 청구 일당) + 직접경비 + 예비비 10%
```

**예시 — 브랜딩 패키지**:
- 대표 5일 × 600,000 = 3,000,000
- 디자이너1 15일 × 400,000 = 6,000,000
- 직접경비(외주 일러스트): 500,000
- 소계: 9,500,000
- 예비비 10%: 950,000
- **견적가 (공급가): 10,450,000원**
- 부가세 10%: 1,045,000
- **총액: 11,495,000원**

### 5.5 견적 작성 체크리스트

- [ ] 투입 인원·일수를 **실측이 가능한 단위**로 산정 (막연한 "총 얼마"가 아니라)
- [ ] 직접경비(인쇄·외주·사진구매 등) **별도 항목으로 명시**
- [ ] 예비비 10% 반영
- [ ] 부가세 별도 명시
- [ ] **수정 범위** 명시 ("시안 2회 + 수정 3회 이내" 등)
- [ ] 초과 시 **추가 비용 공식** 명시 ("추가 수정 1회당 300,000원" 등)
- [ ] 대금 지급 시기 명시 (계약금 50% / 잔금 50% 등)
- [ ] **납기 지연 책임** 조항

## 6. 📌 규칙 #5 — 프로젝트 사후 손익 판정

### 6.1 공식

```
프로젝트 실손익 = 매출(공급가) − Σ(실제 투입일 × 원가 일당) − 실제 직접경비
```

### 6.2 판정 기준

| 조건 | 판정 | 의미 |
|---|---|---|
| 매출 ÷ 총원가 ≥ 2.0배 | 🟢 **이익** | 목표 달성 |
| 1.3배 ≤ 매출 ÷ 총원가 < 2.0배 | 🟡 **본전 근처** | 간접비는 회수, 이익 부족 |
| 매출 ÷ 총원가 < 1.3배 | 🔴 **손해** | 간접비 미회수 |
| 매출 ÷ 총원가 < 1.0배 | 🔴🔴 **순손실** | 인건비도 못 건짐 |

### 6.3 검증 사례 — 대구광역시청년센터 홍보물 (2026.03.12)

**매출**: 750,000원 (총액) → 공급가 **약 682,000원**

**투입 원가**:
| 인원 | 투입 | 원가 일당 | 원가 |
|---|---|---|---|
| 디자이너1 | 1.0일 | 199,300 | 199,300 |
| 대표 | 2시간 (0.25일) | 303,300 | 75,825 |
| 인쇄비 (직접경비) | | | 80,000 |
| **총원가** | | | **약 355,000원** |

**판정**:
- 전체 배수: 682,000 ÷ 355,000 = **1.92배** 🟡 본전 근처
- 서비스 배수 (인쇄 제외, 규칙 #7 관점 B): 602,000 ÷ 275,000 = **2.19배** 🟢 이익
- **실질 판정**: 🟢 **건강한 프로젝트** (인쇄는 통과항목으로 처리)
- 서비스 마진: 약 327,000원

**교훈**:
- 짧고 깔끔한 홍보물 프로젝트는 이 규모에서 건강한 수익
- 단, 대표 "2시간"이 실제로 정확한지는 타임시트로 검증 필요 (견적 작성·인쇄소 연락·파일 전달 시간 포함 시 더 길 수 있음)
- 이 프로젝트가 "괜찮았다"는 것을 숫자로 증명한 첫 사례

## 7. 📌 규칙 #6 — 고객 대응 및 단가 방어

### 7.1 고객이 "단가가 비싸다"고 할 때 표준 답변

> "네, 저희 디자이너의 **인건비 원가**는 일당 약 20만원이 맞습니다.
> 여기에 사무실·장비·소프트웨어·관리 간접비, 수정 대응 비용, 그리고 회사 운영이익이 더해져 **청구 단가**가 40만원입니다.
> 이는 업계 표준 2배수(멀티플라이어 2.0)로, 1인 MM 단가로 환산하면 800만원대입니다.
> 이는 국내 디자인 에이전시 MM 단가 범위(800만~1,500만)의 **가장 낮은 축**입니다."

### 7.2 "원가와 가격은 다르다" 설명용 비유

- **아메리카노**: 원두 원가 500원 / 판매가 5,000원 (10배)
- **햄버거**: 패티 원가 1,500원 / 판매가 9,000원
- **의류**: 공장 출고가 20,000원 / 매장 80,000원

**원가를 구매하려면 직접 고용해야 한다.**
고객이 디자이너를 직접 고용할 경우 실제 월 부담: 월급 230 + 4대보험 23 + 퇴직 19 + 장비·공간·SW 40 + 관리·교육·채용비 = **월 500~700만원** + 비가동·휴가·퇴사 리스크.

### 7.3 🚨 수상한 고객 판별

**다음 고객은 받지 않는다**:
- 원가 공개를 요구하면서 마진을 부정하는 고객
- "직원 월급이 얼만데 왜 그만큼이냐"고 따지는 고객
- 견적 단가를 경쟁사와 직접 비교하며 무한 네고하는 고객
- 업계 단가를 부정하며 "특별 할인"을 요구하는 고객

**이유**:
- 무한 수정을 요구함 (원가로만 팔면 추가 요청이 당연해짐)
- 대금 지급이 늦음 (을 취급)
- 재의뢰가 없음 (싸게 쓰는 게 목적)
- 업계에 "거기 40만원이래"라고 역으로 소문냄

**건강한 고객은 단가를 묻지 않고 결과물의 가치를 평가한다.**

## 8. 📌 규칙 #7 — 직접경비(통과항목) 처리

### 8.1 직접경비란

고객 프로젝트를 위해 외부에 지출하고 그대로 청구하는 비용:
- 인쇄비
- 외주 일러스트·촬영·번역·개발
- 사진·폰트·소스 구매
- 택배·퀵서비스
- 출장 교통비·숙박비
- 행사 대관료·장비 렌탈

### 8.2 처리 원칙

1. **견적서에 별도 항목으로 명시** — 서비스 단가에 숨기지 않기
2. **원가 그대로 청구** 또는 취급수수료 10% 얹기 (업계 관행)
3. **손익 판정 시 매출·원가 양쪽에서 동일금액 제외** 후 "서비스 배수"로 판단
4. **영수증 무조건 보관** — 부가세 매입공제 대상
5. **고객 카드 결제 유도 가능** — 대형 인쇄·렌탈은 고객 직접 결제가 현금흐름상 유리

### 8.3 손익 판정 시 관점 분리

```
전체 배수 = 매출 ÷ (인건비 원가 + 직접경비)
서비스 배수 = (매출 − 직접경비) ÷ 인건비 원가   ← 실질 판단용
```

## 9. 📌 규칙 #8 — 수주 단계 "투입 한계 일수" 계산 (가장 실전적)

### 9.1 왜 이 규칙이 필요한가

과거 프로젝트를 소급 계산하려 해도 **시간 기록이 없어 불가능**하다. 또한 여러 프로젝트를 동시에 진행하기에 사후 배분도 어렵다. 따라서 나디는 **견적 단계에서 "이 금액이면 며칠 안에 끝내야 하는지"를 먼저 확정**하는 방식으로 손익을 관리한다.

### 9.2 핵심 공식

```
순 매출 = 매출(공급가) − 직접경비

🟢 이익 달성 인건비 예산 = 순 매출 ÷ 2.0
🟡 본전 인건비 예산     = 순 매출 ÷ 1.3
🔴 순손실 경계 예산     = 순 매출 ÷ 1.0

투입 가능 일수 = 인건비 예산 ÷ 원가 일당
```

### 9.4 혼합 투입 계산 (대표 + 디자이너)

대부분의 프로젝트는 대표가 일부 투입된다. 계산 순서:

```
1단계: 순 매출 ÷ 2.0 = 이익 달성 인건비 예산
2단계: 이익 예산 − (대표 투입일 × 303,300) = 디자이너 가용 예산
3단계: 디자이너 가용 예산 ÷ 디자이너 원가 일당 = 디자이너 최대 투입 가능 일수
```

**예시 1 — 200만 프로젝트, 대표 1일 + 디자이너1**
- 이익 예산: 200만 ÷ 2.0 = 1,000,000
- 대표 1일 차감: −303,300
- 디자이너1 남은 예산: 696,700
- 디자이너1 최대 투입일: 696,700 ÷ 199,300 = **3.49일**
- 🟢 결론: **대표 1일 + 디자이너1 3.5일 이내 완료 시 이익**

**예시 2 — 500만 프로젝트, 대표 2일 + 디자이너1**
- 이익 예산: 2,500,000
- 대표 2일: 606,600
- 디자이너1 예산: 1,893,400
- 디자이너1 최대 투입일: **9.50일**
- 🟡 본전선: 대표 2일 + 디자이너1 **16.32일**
- 결론: 🟢 9.5일 이내 / 🟡 16.3일 이내 / 🔴 이후는 손실

**예시 3 — 1,000만 프로젝트, 대표 5일 + 디자이너 2명 공동**
- 이익 예산: 5,000,000
- 대표 5일: 1,516,500
- 디자이너 공동 예산: 3,483,500
- 두 디자이너 평균 원가 약 208,000/일 → 합계 **약 16.75일**
- (예: 디자이너1 8일 + 디자이너2 8.75일)

### 9.5 직접경비 반영

**반드시 매출에서 먼저 빼고 예산 계산**:

```
순 매출 = 공급가 − (인쇄비 + 외주비 + 사진구매 + 택배 + 기타 프로젝트 전용 지출)
```

**예시 — 500만 프로젝트, 인쇄비 80만 예정**
- 순 매출: 5,000,000 − 800,000 = 4,200,000
- 이익 예산: 2,100,000
- 대표 1일: 303,300 → 남은: 1,796,700
- 디자이너1 최대 투입일: **9.02일**

### 9.6 수주 판단 3단계

| 단계 | 행동 |
|---|---|
| 1 | 프로젝트 범위 기반 **예상 공수** 산정 (누가 며칠 들어갈지) |
| 2 | 위 공식·테이블로 **한계선** 확인 |
| 3 | 예상 공수 vs 한계선 비교 → 아래 판정 |

| 조건 | 판정 |
|---|---|
| 예상 공수 < 🟢 이익선 | ✅ **수주** (여유 있음) |
| 🟢 ~ 🟡 사이 | ⚠️ **수주하되 타이트하게 관리** (일정 초과 즉시 손익 악화) |
| 🟡 ~ 🔴 사이 | 🚨 **단가 협상 필수** 또는 거절 |
| 🔴 초과 | ❌ **수주 거절** 또는 범위 축소 협상 |

## 부록 A — 용어 정리

| 용어 | 뜻 |
|---|---|
| **원가 일당** | 직원 한 명이 프로젝트에 하루 투입될 때 회사가 실제로 지출하는 금액 (급여+4대보험+퇴직금 ÷ 실근무일 ÷ 가동률) |
| **청구 일당 (Billing Rate)** | 고객에게 청구하는 하루 단가 = 원가 일당 × 멀티플라이어 |
| **멀티플라이어** | 원가 대비 청구단가 배수. 표준 2.0 |
| **가동률 (Utilization)** | 출근시간 중 실제 프로젝트 작업에 쓰는 비율. 나디 기준 65% |
| **MM (Man-Month)** | 1인이 한 달(보통 20~22일) 투입할 때의 인건비 단위 |
| **간접비** | 프로젝트와 무관하게 매달 나가는 고정비 (임대료·SW·세무 등) |
| **직접경비** | 특정 프로젝트를 위해 외부에 지출하는 비용 (인쇄·외주 등) |
| **BEP (Break-Even Point)** | 손익분기점. 이익도 손해도 없는 매출 수준 |
| **공급가** | 부가세 제외 금액. 매출·손익 판정은 항상 공급가 기준 |
| **감가상각** | 장비를 몇 년에 걸쳐 나눠서 비용 처리하는 방식. 나디는 5년(60개월) 기준 |
| **통과항목 (Pass-through)** | 회사가 마진 없이 그대로 고객에게 청구하는 직접경비 |

## 부록 B — 자주 틀리는 지점

1. **"이번 달 통장에 돈이 있으니 이익"** — ❌ 부가세·세금·다음 달 지출이 쌓여있음
2. **"월급보다 단가가 훨씬 높으니 이익"** — ❌ 간접비·가동률·퇴직금을 계산해야 함
3. **"인쇄비도 마진 계산에 포함"** — ❌ 통과항목은 서비스 배수에서 빼야 정확
4. **"대표는 월급 안 가져가니 원가 제로"** — ❌ 대표 시간도 원가. 안 가져가면 그만큼 회사에 빚이 쌓이는 것
5. **"견적은 감으로"** — ❌ 규칙 #4 공식 없이 견적하면 손해 프로젝트를 수주함
6. **"총액 기준으로 손익 계산"** — ❌ 공급가 기준으로 해야 부가세 혼동 없음
7. **"단가 낮춰서라도 수주"** — ❌ 멀티플라이어 1.5 미만은 간접비도 못 건짐
